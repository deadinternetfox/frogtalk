Welcome

FrogTalk is a federated chat app. Every account lives on a node — a server you trust. Most people use the official node; you can point the app at your own self-hosted server too.

  • End-to-end encrypted DMs on every node
  • Voice & video calls with safety numbers
  • Switch nodes anytime in Settings → Network

Pick your node

Official frogtalk.xyz is pre-filled — tap a chip or edit the address for a community or self-hosted node.

Enter a valid hostname (e.g. frogtalk.xyz or your.node.com).

Self-hosting? Run the node setup wizard on your server — see docs/node on desktop, or your operator’s invite link.
